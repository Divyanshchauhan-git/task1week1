# Trip Management System

A Full Stack Trip Management System built using:

- React
- FastAPI
- PostgreSQL
- SQLAlchemy

## Features

- User Signup
- User Login
- Persistent Authentication
- Create Trips
- View Trips
- Update Trips
- Delete Trips
- PostgreSQL Database Integration

## Frontend

Built using React and Fetch API.

## Backend

Built using FastAPI and SQLAlchemy ORM.

## Database

PostgreSQL relational database.

## Run Frontend

```bash
npm run dev
```

## Run Backend

```bash
uvicorn app.main:app --reload
```
